INSTALLING DEVKNOX

Step 1

In the Intellij IDEA IDE select File menu and choose Settings(Ctrl+Alt+S).

Step 2

On the left hand pane select Plugins.

Step 3

On the Plugins page that opens in the right-hand part of the dialog, click Install plugin from disk.

Step 4

On the file explorer menu traverse or type the path to the plugin file which is the archive file. Select the plugin archive & click Ok.

Step 5

Click on Apply and check if Devknox is installed and checked on the Plugins Page menu.

Step 6

Click ok. The IDE will propmpt for a restart. Restart the IDE. After restart Devknox is added to the menu bar.

USING DEVKNOX

Step 1

On the menu bar click on Devknox and select Login.

Step 2

On the login dialog box provide the credentials and click Login.

Step 3

After succesfull login to Devknox select your module in the project pane which is in the left side of the IDE. Right click on selected module and select Devknox Scan->Scan Module.

Step 4

If the report window does'nt pop up automatically, go to View->Tool Windows and select Devknox Report: The Report Window has two parts. The left part shows the issues in a tree structure and the right part shows the overview of the issues.

Step 5

Expand the tree structure to see all the issues. Click on the issue to see the details of the issue like the location, suggestion etc on the right pane of devknox report window.

Step 6

Double Click on an issue on the left part of devknox report window to go to the exact location of the issue on the IDE. The issue is marked with an mine icon on the gutter area.

Step 7

Use the intention(bulb icon) to fix the issue automatically.

Step 8

Select Enable Devknox on Devknox menu to toggle viewing issue locations.

Step 9

Select Logout on Devknox menu to logout from Devknox which disables functionality of Devknox.
